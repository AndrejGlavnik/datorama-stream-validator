# Data Stream Naming Conventions

## Stream Name Pattern

```text
COUNTRY_CATEGORY_SOURCE_DESCRIPTION
```

Example:

```text
NL_CORE_GA4_WEB_EVENTS
```

## Campaign Name Pattern

```text
COUNTRY_CATEGORY_CHANNEL_DESCRIPTION
```

Example:

```text
NL_CORE_SEARCH_BRAND
```

## Governance Notes

- Country should use a two-letter uppercase market code.
- Category should match the approved taxonomy.
- Source should match approved source labels, not platform nicknames.
- Exceptions should be documented, owned and reviewed before dashboard refreshes.
